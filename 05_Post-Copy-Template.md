> A carousel without post copy is half-finished. This file ships **with** the PDF and PPTX.
>
> Fill in each section. Save the final version as `output/<TopicSlug>_post-copy.md`.

---

## Hook line (visible above the "see more" fold — first ~140 chars)

> `[One line that earns the click. Tease the carousel without giving away the punchline.]`

**Self-check:**
- [ ] Under 140 characters
- [ ] No emojis
- [ ] Standalone — works without seeing the rest of the post
- [ ] Specific, not generic ("We rebuilt onboarding in 30 days" beats "Some thoughts on onboarding")

---

## Body (200–1300 chars total including hook)

```
[3–6 short paragraphs. Plain language. Short sentences.

Set up the carousel — what's it about, why now?

Name the most surprising insight from inside the carousel.

End with a question or invitation that connects to the comment-bait closer below.]
```

**Self-check:**
- [ ] 200–1300 chars total
- [ ] 3–6 short paragraphs (1–3 sentences each)
- [ ] No "🔥 thread 🔥" / "Let me explain ↓" tells
- [ ] No links (links go in the first comment)
- [ ] Mentions any co-presenters, photographers, or collaborators by handle

---

## Comment-bait closer (1–2 lines)

> `[One specific question that's easy to answer in a comment.]`

**Self-check:**
- [ ] Specific, not "What do you think?" — e.g., "Which prompt are you trying first — tactical or stretch?"
- [ ] Easy to answer in 1–2 sentences
- [ ] Connects to a slide inside the carousel

---

## Hashtags (3–5 max, brand-led, never generic)

`#[Tag1] #[Tag2] #[Tag3]`

**Allowed:** product/method/brand names — `#Copilot`, `#M365`, `#YourFrameworkName`
**Never:** `#motivation`, `#business`, `#leadership`, `#tech`, `#thoughtleadership`, `#hustle`

**Self-check:**
- [ ] 3–5 hashtags max
- [ ] All brand-led (tools, methods, named frameworks)
- [ ] Consistent casing across posts (lower-case OR PascalCase, not mixed)

---

## First-comment link (post separately as the first reply on LinkedIn)

> `[Link to the carousel's deeper resource — the source deck, a blog post, the AI assessment, etc. + one short sentence framing it.]`

---

## Pre-publish checks

- [ ] Hook line is the whole game — would I click on this if I'd never heard of the topic?
- [ ] No emojis in the first line
- [ ] No links in the body — link is queued for the first comment
- [ ] Hashtags brand-led
- [ ] Tagged co-presenters / photographers if relevant
- [ ] Specific ask, not "Hope this helps!"
