*Template by Ashley Pyle, GadellNet · [linkedin.com/in/ashleypyle](https://linkedin.com/in/ashleypyle) · adapt freely for your brand.*

> Run this before declaring the carousel done. Any unchecked box on a BLOCKING line means you don't ship.

**Author:** `[Name]`

---

## 1. Format

- [ ] All 10 slides use only your three locked background colors (DARK / LIGHT / CTA)
- [ ] Every card has the locked edge frame (1.5pt TINT) — no bare cards
- [ ] Every photo has the edge frame; headshots are circle-cropped
- [ ] No font below mobile minimums (body ≥ 14pt, captions ≥ 11pt)
- [ ] No slide exceeds the density cap (BLOCKING)
- [ ] No card grid contains more than 4 tiles
- [ ] Photos pre-cropped to box ratio — no skew
- [ ] **No accent lines under titles** (eyebrow blocks only) (BLOCKING)
- [ ] Logo on slide 1 only

## 2. Content

- [ ] Source-deck preflight complete — every fact traces to source (BLOCKING)
- [ ] No invented stats, names, quotes, or photo credits (BLOCKING)
- [ ] Cold-reader test passes on every closer
- [ ] Slides 8 + 9 differentiate by ask (tactical vs stretch)
- [ ] No footers anywhere (no author byline, no page numbers)
- [ ] Swipe cue on slides 1–9 (not 10)
- [ ] Corner accent on every slide
- [ ] Photo credits present on every third-party image

## 3. Accessibility (WCAG 2.1 AA — BLOCKING)

- [ ] All text/background pairings hit 4.5:1 (body) or 3:1 (large)
- [ ] Every image has alt text (or `alt=""` if decorative)
- [ ] No information conveyed by color alone
- [ ] CTA mechanism (QR code or similar) ≥ 1.8" square
- [ ] Reading order is top→bottom, left→right
- [ ] Body copy at Flesch reading ease 60+ (8th-grade level or below)

## 4. CTA (slide 10)

- [ ] Eyebrow present (`ONE ASK` or your locked phrase)
- [ ] Three-line hero is your locked phrasing — not a one-off
- [ ] Author byline present (name + role + one-line credibility)
- [ ] CTA mechanism (QR / link / sign-up) is the locked asset, not a substitute
- [ ] Three actions stacked correctly (SAVE / COMMENT / FOLLOW)
- [ ] Logo top-right, white-on-dark variant

## 5. Post Copy (BLOCKING — must ship with the carousel)

- [ ] Hook is in the first ~140 characters
- [ ] No emojis in the first line
- [ ] Body is 200–1300 chars total
- [ ] 3–6 short paragraphs
- [ ] One specific comment-bait question
- [ ] 3–5 hashtags, brand-led, no generic ones (`#motivation`, `#business`, `#leadership`)
- [ ] No links in the post body (link goes in the first comment)

## 6. Delivery

- [ ] Final PDF in `output/<TopicSlug>_LinkedIn_Carousel.pdf`
- [ ] PPTX source in `output/<TopicSlug>_LinkedIn_Carousel.pptx`
- [ ] Post copy file in `output/<TopicSlug>_post-copy.md`
- [ ] PDF metadata (Title, Author, Subject) verified

## 7. Final Read-through

Read the deck once on your phone (LinkedIn preview, if possible). Ask:

- [ ] Does slide 1 stop the scroll?
- [ ] Does each slide earn its swipe?
- [ ] Is there exactly one ask?
- [ ] Would a stranger get the point?

If the answer to any of these is "no," go back to the relevant step in the SOP.
